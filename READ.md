# Local Cloud Lab

Proyecto de aprendizaje para montar en local una pequeña plataforma con:

- Terraform
- Kubernetes local con kind
- Kong como API Gateway
- MinIO como almacenamiento S3-compatible
- Un microservicio backend
- Un frontend que consume el backend

## Objetivo

Aprender paso a paso infraestructura como código, despliegue local y comunicación entre servicios, documentando todo el proceso.

## Roadmap inicial

- [ ] Crear clúster local con kind
- [ ] Conectar Terraform con Kubernetes

- [ ] Desplegar un frontend simple
